Principais formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag <b> (tag não semântica, ou seja, ela não tem significado apenas formato em negrito)

Nesta frase, temos um termo em destaque usando a tag <strong> (tag semântica, com significado)

Visualmente essas duas frases não possuem diferença, mas a formatação <b> só tem sentido de forma; enquando a tag <strong>, além da forma atribui um sentido para a frase, porque nessa frase o termo é forte, de destaque, strong.

Itálico / Ênfase

Nesta frase, temos um termo em Itálico usando a tag <i> (tag não semântica)

Nesta frase, temos um termo em ênfase usando a tag <em> (tag semântica)

Novamente, visualmente as duas frases não possuem diferença, mas a formatação <i> só tem sentido de forma; enquanto a tag <em> tem significado.

Tags que serão obsoletas

As tags <b> e <i> pertencem ao documento HTML4, ainda não são consideradas obsoletas, mas a tendência é que em algum momento elas passem a serem inutilizáveis, impróprias para o uso em HTML5, por isso não é recomendado o uso. Então sempre que quiser destacar ou enfatizar uma frase use respectivamente: <strong> e <em>

Negrito / Itálico = Forma = CSS
Destaque / Ênfase = sentido = HTML5

Texto marcado

Podemos também criar um texto marcado usando a tag <mark>

Em outro parágrafo, temos outro texto marcado no final.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.

Enquanto a tag para aumentar as letras <big> é considerada obsoleta por ser uma forma, a tag para diminuir <small> atribui sentido (como por exemplo: para ler as letras pequenas de um contrato) por isso essa tag não é vista como obsoleta.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado (tag <del>).

Texto inserido

Podemos marcar um texto como um texto inserido para dar uma ênfase e indicar que ele foi adicionado depois (tag <ins>).

Existe também o sublinhado usando a tag <u> (tag não semântica)
